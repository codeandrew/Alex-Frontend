# LEGG APP 
> LEGG Mobile UI

## First install
- npm install
- npm install -g live-server

## Run SASS and Dev server
```
npm run sass
```
> We run the sass script.
```
live-server --open=public
```
> Finally, we start the development server.

### Dependencies
- Bootstrap 4
- Jquery
- Popper.js
